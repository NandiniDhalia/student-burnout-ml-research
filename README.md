# Predicting Student Burnout Using Social Media Usage Patterns and Behavioral Indicators: A Machine Learning Approach

## Overview

Student burnout has become an increasingly important concern due to rising academic pressure, lifestyle changes, and extensive social media usage. This research project aims to analyze the relationship between social media usage patterns and behavioral indicators in students and develop machine learning models for burnout prediction.

The project focuses on identifying major factors contributing to burnout and comparing the effectiveness of different machine learning algorithms for predictive analysis.

---

## Research Objectives

- Analyze the impact of social media usage patterns on student burnout.
- Identify behavioral indicators associated with burnout.
- Perform exploratory data analysis to uncover trends and relationships.
- Build and compare machine learning models for burnout prediction.
- Evaluate model performance using appropriate metrics.

---

## Research Questions

**RQ1:** How do social media usage patterns relate to student burnout?

**RQ2:** Which behavioral indicators contribute most significantly to burnout?

**RQ3:** Which machine learning model predicts student burnout most effectively?
